What is this?

This is a basic first person raycaster game whose engine is completely built in a GGUF file run by a standard ollama server. The GGUF was not obtained by training, but hand crafted (I mean, figuratively) to handle the game.

Every keypress the frontend sends a command to the server, and it gets as response 35 tokens: the new game state, and 32 columns that tell it what to render and where. The context has no history, the current state keeps getting computed from the whole "conversation".
