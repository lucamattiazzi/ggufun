GET READY FOR THE BUBBLE BURSTING!

▪ a catalog of ideas to reuse all those unused Ollama servers ▪

A lot of people always knew: AI is a fad. At some point, the bubble will burst, and all those Ollama servers built during the past years will remain unused, littering the web with their gray chimneys, the immense models they used to run forever gathering e-dust.

But you can still use them! Why, the GGUF files they used to load were humongous models, trained on the whole human knowledge, but a GGUF can be so much more! This is a catalog of "useful" GGUF files to be run on your Ollama server RIGHT NOW: nothing trained, nothing "grown", only hand crafted, single use collection of tensors, to give back a purpose to what will become, some day, e-waste.

How to read a plate. Choose any machine from the list on the left: on each page you can test the machine, check all the information on it, take the GGUF, or see how it was built.
